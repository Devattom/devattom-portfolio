Pokedex.

An Angular project.

_This application is made with Angular 15.

_This project was made to learn the basics of the framework.

_First you have a form connection made to use guards of Angular.

Add pokemon.

_You are able to add pokemon, just keep in mind that the data are not stored, it is only a front-end project.

_You can choose a name, an image thanks to the pokemon API (you just have to change the 'xxx' by the number of the pokemon), the HP and the CP and then max 3 types.

Detail and edit.

_By clicking on a pokemon card you have access to his details.

_From here you are able to delete it, or edit it.

_You can edit a pokemon.

_You can change, the name, HP and CP but also the ty.

Search.

_The search bar allow you to find any pokemon.
