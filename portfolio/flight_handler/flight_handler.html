Booking.

List of flight.

_This application is made with Symfony 6.

_This project was made to learn the basics of the framework.

_You have the possibilitie to look the list of flights when you are not connected, to connect or to create an account.

Connect.

_There is 2 possibilities, you have an account and then you can connect yourself, or you don't have and you need go trought the registration form.

_When you are connected, differents levels are available, the user level, the common one, the one you will get after your registration, and the admin one. If you want to test the admin user session use the following credentials (test2@test.com, test123)

Normal user VS admin user.

_When connected in admin session the interface will change and allow you to do more.

_You will be able to create a new city.

_Also create new flights, the number of the flight will be automaticly and randomly generate everytime.

_After all the data validation, all the information will be stored in the database.

_And then the flight list will let you administrate everything from this page, You can see details of flight, you can edit a flight and even delete it.
