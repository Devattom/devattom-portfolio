List of flight.

_This application is made in pure PHP, it is a litle project made to learn PHP with OOP development.

The application propose you a list of fake hotels.

_All the hotels are stored in a database, with an exact number of rooms dedicated to each hotel.

_When you click on the booking link you will navigate to the form reservation, and the hotel you have choose will be automaticly selected.

Connect.

_The form reservation will ask you multiple informations, name, mail, hotel desired, start date and end date reservation. Everything will be stored in the database.

_A room will be assigned directly to the reservation if no room is available for the selected date an error message will be shown.

_A reservation cannot be taken in the past.

Normal user VS admin user.
